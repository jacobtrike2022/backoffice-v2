# Favicon Setup

This directory should contain the following favicon files:

- `favicon.ico` - Main favicon (16x16, 32x32, 48x48 sizes recommended)
- `favicon-32x32.png` - 32x32 PNG favicon
- `favicon-16x16.png` - 16x16 PNG favicon  
- `apple-touch-icon.png` - 180x180 PNG for iOS devices

## Adding Your Favicon

1. Place your favicon files in this `public/` directory
2. The `index.html` file is already configured to reference these files
3. For best results, use an orange unicycle icon that matches your brand

## Favicon Generator Tools

If you need to generate multiple sizes from a single image:
- https://realfavicongenerator.net/
- https://favicon.io/
